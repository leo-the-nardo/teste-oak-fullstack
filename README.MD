# Desafio OAK
![img_1.png](img_1.png)
https://github.com/leo-the-nardo

Tecnologias utilizadas:
- Java - Spring (Backend)
- Typescript/Javascript - NextJS (Frontend)
- PostgreSQL - Banco de Dados
- Docker


## Run
```sh
docker compose up -d
```
Pronto !

## Acessar em:

```sh
http://localhost:3000/
```
## Documentação da API disponivel:

```sh
http://localhost:8080/swagger-ui/index.html#/
```
